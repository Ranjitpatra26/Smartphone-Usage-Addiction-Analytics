# 📱 Smartphone Usage & Addiction Analytics Dashboard using Power BI

## 📌 Project Overview

This project analyzes smartphone usage patterns, social media behavior, addiction levels, and health indicators using **Python (Pandas)** for data preprocessing and **Power BI** for interactive dashboard development.

The dashboard combines two datasets to provide comprehensive insights into smartphone addiction, screen time, mental health, sleep patterns, and social media usage through interactive visualizations, KPIs, slicers, and navigation.

---

# 🎯 Objectives

- Analyze smartphone usage behavior
- Study social media platform preferences
- Measure smartphone addiction levels
- Evaluate the impact on mental health
- Analyze sleep patterns and stress levels
- Compare addiction across academic levels
- Generate data-driven insights and recommendations

# 📈 Key Insights

- 📱 Higher screen time is strongly associated with higher addiction levels.
- 😴 Students with poor sleep tend to spend more time on smartphones.
- 🧠 Mental health scores decrease as smartphone addiction increases.
- 📚 Graduate students recorded the highest average mental health score.
- 📉 High school students showed comparatively lower mental health scores.
- 📲 WhatsApp and Instagram are among the most preferred social media platforms.
- ⚠️ High stress users spend more time on smartphones than low stress users.
- 💡 Balanced smartphone usage contributes to improved sleep quality and better mental well-being.
